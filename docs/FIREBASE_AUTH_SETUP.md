# Configurazione Firebase Auth (Email/Password)

Per usare il login con email e password:

1. Vai su **Firebase Console** → il tuo progetto
2. **Authentication** → **Sign-in method**
3. Abilita **Email/Password** (Email/Password provider)
4. Salva

Dopo aver abilitato, l'app supporterà:
- **Accedi** – login con email e password
- **Registrati** – creazione nuovo account
- **Ricordami** – salvataggio sicuro per accesso automatico al prossimo avvio
